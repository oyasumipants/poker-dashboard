ポケットエースは最強のスターティングハンド。BTNの3betに「コール」したのは、COに再レイズ(スクイーズ)させるための罠(トラップ)。COのQQがオールインしてきたところを喜んでコール。文句なしのプレイ。

ストリート判断コメント
3bet (6.9bb)良いKQsはHJから3betする十分な強さ
4betにCall分岐点UTGの4betレンジ(AA/KK/QQ/AK)に対してKQsはほぼ全敗。フォールドが無難
フロップCall仕方ないKヒットで降りるのは難しい。ただ相手AKなら既に負けている
ターンCall疑問相手が3連続ベット = 超強い。ここで降りる選択肢もあった
リバーCall悪い3ストリート全ベットに全コールは、相手がAK以上を持っていることを無視している

核心: 「当たっている」と「勝っている」は別物。フロップでKが落ちてKペアが完成しても、4betポットでは相手もKを持っている(AK)か、それ以上(AA/KK)の可能性が非常に高い。

教訓: 4betされたらKQは基本フォールド。

BTN A♠4♦ -20bb 弱いハンドで深入りしすぎ

ヒーロー: BTN  A♠4♦
相手: Leeeeeee(CO) 7♥7♦

【Preflop】HJ Raise → CO Call → ヒーロー Call → BB 3bet 6bb → 全員Call (4way)

【Flop】A♥ 7♠ 6♣ → CO Bet 14bb → ヒーロー Call
【Turn】K♥ → CO Bet 17.5bb → ヒーロー Fold

A4oはBTNでも強くない。BBの3bet(6bb)に4wayでコールするのが問題。Aがヒットしてもキッカーが4と低いので、AKやAQに負けるパターンが多い。さらに4人参加のポットでは誰かがツーペアやセット(今回相手は77のセット)を持っている確率が高い。

教訓: Aがあってもキッカーが弱い(A2〜A5)は、多人数ポットでは慎重に。

UTG K♥T♠ -3bb UTGリンプの問題

ヒーロー: UTG  K♥T♠

【Preflop】ヒーロー Call (リンプ) → HJ Raise → ヒーロー Call
【Flop】6♣ 5♣ 4♣ → 何もヒットせずFold

UTGからリンプ(レイズせずコール)するのは弱いプレイ。KToはUTGから参加するには微妙なハンドで、参加するならレイズ、でなければフォールドが基本。

教訓: 「レイズする勇気がないハンドはそもそも参加しない。」

具体的アドバイス

1. 「4betされたらKQは降りる」ルール

これ1つで今回のセッションは+88bbのプラスだった。相手の4betはAA/KK/QQ/AKを示す超強いサイン。3betは良いプレイだが、4betが返ってきたらフォールドが正解。

2. BTNでも「A+小さいカード」は多人数ポットを避ける

A4oは1対1なら悪くないが、3人以上参加するポットではキッカー負けしやすい。BTNは有利なポジションだが、ハンド選びは大事。

3. 「レイズかフォールド」を徹底

リンプは1回だけで前回より改善。「参加するならレイズ、自信がなければフォールド」を引き続き意識。特にSBからのコールだけの参加は避ける。

4. BB防御をポットオッズで判断する

「なんとなく弱そう」ではなく、ポットオッズに見合うかで判断する。2-3bb程度のレイズなら、J9o/KTo/87s等でもコールして守った方がお得な場面が多い。目標は防御率40-50%。

総括

AAで+123.75bbの大勝ちがあったにも関わらず、KQoの4betコールで-100bbが響いてセッション全体は-11.8bb。UTG引き締め、CO改善、CBet率向上、3bet率向上など前回からの成長は多いが、「相手が本当に強い時にいつ降りるか」が次の課題。特に4betポットでの立ち回りが今後の重要テーマ。
